# C. 可执行 PRD（供 Replit 直接实现）

## 1. 产品目标 & KPI

* **目标**：在“nano banana 3d figurines”与相关长尾上，8–12 周内争取 Google Top5。
* **核心 KPI**：

  * SEO：≥40 可索引页、非品牌自然 UV ≥3k/月、主落地页 CTR ≥8%。
  * 功能：出图成功率 ≥95%，首图 <12s；**STL 导出转化 ≥8%**；打印报价点击率 ≥3%。

## 2. 用户画像 & 关键场景

* **宠物主**：把狗/猫做成“手办风”，最好能**打印**。
* **情侣/礼物需求**：希望**一键模板**产出可复现风格 + 快速下单。
* **二次元/玩家**：要**包装盒/台座**等“陈列感”。
* **入门用户**：不懂 3D，只要**图→STL**自动完成。

## 3. 信息架构（IA）

* 顶部导航：**Generator**｜**Make it 3D (STL)**｜**Print My Figurine**｜**Prompt Packs**｜**Learn**
* 关键 URL：

  * `/nano-banana-3d-figurines`（**主落地页**）
  * `/generator`（手办风生成器）
  * `/make-it-3d`（图→网格→**STL 导出**）
  * `/print`（Craftcloud/Shapeways **即时报价/下单**）
  * `/prompts/*`（主题化包） `/guides/*`（HowTo/FAQ/打印指南）
    （主落地页与指南页加 **HowTo/FAQPage Schema**；工具页加 **SoftwareApplication + ImageObject**。）

## 4. 核心功能

**F1｜手办风生成器（必须）**

* 上传照片/URL → 选择主题模板（Pet/Anime/Couple/Office Desk/Packaging 等）→ 生成 2–4 张 → 选图下载。
* **调用 Gemini 2.5 Flash Image（Nano Banana）**；显式声明输出图含 **SynthID 水印**（FAQ 链接）。([谷歌开发者博客][5])
* **稳定 Prompt 模板**（示例，英文，可复现）：

  > *Create a 1/7-scale commercialized figurine of the person in the photo, realistic lighting, displayed on a computer desk. Use a round transparent acrylic base with no text. Show the ZBrush modeling process on the monitor. Place a BANDAI-style toy box printed with the original artwork next to the monitor.*
  > （此类 prompt 在媒体与社区教程大量出现与验证。）([mint][8])

**F2｜Make it 3D (STL)（王牌差异化）**

* 选择已生成图片 → **TripoSR / Meshy / Tripo API** 单图重建为 3D 网格（glb/obj）→ 自动修复（非流形、法线、最小壁厚）→ **导出 STL**。([Replicate][7])

**F3｜Print My Figurine（即时报价/下单）**

* 上传 STL → **Craftcloud 或 Shapeways API** 实时价格与材料（建议默认：树脂 SLA + 透明亚克力底座），可跳转下单或代打。([Craftcloud®][9])

**F4｜Prompt Packs（主题模板库）**

* **宠物（狗/猫）**、**情侣/婚礼**、**动漫/手办盒**、**办公桌/玩家布置**、**超级英雄风**等；配**失败修复提示**（手指/姿态/光照）。
* 每个模板页提供：示例图、Prompt、参数、常见失败与修复。

**F5｜Learn（指南/FAQ/博客）**

* How-To（AI Studio / Gemini App 路径、操作步骤）。([Beebom][4])
* FAQ：**是否免费/次数**、**SynthID 水印**、**隐私/安全**、**能否 3D 打印**、**商用限制**。([Zee News][10])

## 5. 技术实现（与你现有栈对齐）

* 前端：**React 18 + TypeScript + Vite + Tailwind + Shadcn**（首屏 SSR/SSG 以可索引）。
* 后端：**Express.js + Node.js**。
* 队列：BullMQ（生成/三维转换异步任务）。
* 存储：S3/R2（图像 WebP/AVIF 转码 + CDN）。
* 数据库（Drizzle + Postgres）：`users, jobs, images, meshes, prompts, orders, logs`。
* 关键 API：

  * `POST /api/generate-image` → Gemini 2.5 Flash Image（Nano Banana）。([Google AI for Developers][11])
  * `POST /api/image-to-3d` → TripoSR/Meshy/Tripo 托管 API（返回 glb/obj/stl）。([Replicate][7])
  * `POST /api/print-quote` → Craftcloud/Shapeways 报价/下单。([Craftcloud®][9])
* 成本策略：首期使用 **Replicate / FAL / Tripo API** 计次；量大后自托管 **TripoSR**。([Replicate][12])
